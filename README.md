# Usage
<!-- usage -->
```sh-session
$ npm install -g fisga
$ fisga COMMAND
running command...
$ fisga (--version)
fisga/0.0.2 linux-x64 node-v20.18.1
$ fisga --help [COMMAND]
USAGE
  $ fisga COMMAND
...
```

## `fisga`

Update installed plugins.

```
USAGE
  $ fisga  [-h] [-v]

FLAGS
  -h, --help     Show CLI help.
  -v, --verbose

DESCRIPTION
  Update installed plugins.
```
